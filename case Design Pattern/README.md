# Design Pattern

## 项目说明

Java 设计模式示例集合，演示 23 种经典设计模式。

## 设计模式分类

### 创建型模式

| 模式 | 说明 |
|------|------|
| 单例模式 | 确保一个类只有一个实例 |
| 工厂方法 | 定义创建对象的接口 |
| 抽象工厂 | 创建相关对象的家族 |
| 建造者模式 | 分步骤构建复杂对象 |
| 原型模式 | 通过复制创建对象 |

### 结构型模式

| 模式 | 说明 |
|------|------|
| 适配器模式 | 转换接口 |
| 桥接模式 | 分离抽象与实现 |
| 组合模式 | 树形结构 |
| 装饰器模式 | 动态添加功能 |
| 外观模式 | 简化子系统接口 |
| 享元模式 | 共享细粒度对象 |
| 代理模式 | 控制对象访问 |

### 行为型模式

| 模式 | 说明 |
|------|------|
| 责任链模式 | 请求链式处理 |
| 命令模式 | 封装请求 |
| 迭代器模式 | 顺序访问集合 |
| 中介者模式 | 解耦多对象交互 |
| 备忘录模式 | 保存和恢复状态 |
| 观察者模式 | 一对多依赖 |
| 状态模式 | 状态决定行为 |
| 策略模式 | 可替换算法 |
| 模板方法 | 定义算法骨架 |
| 访问者模式 | 分离操作与结构 |
