| Spring Boot | 2.7.18 |

## 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd case
```

### 2. 安装父 POM

```bash
cd "case 0 All Parent/com-gang-case-parent"
mvn install
```

### 3. 运行示例

```bash
cd "case 1.1 Base Application"
mvn spring-boot:run
```

## 注意事项

1. 部分项目需要外部服务（MySQL、Redis、RabbitMQ 等）
2. 数据库相关项目请先配置数据库连接
3. 每个项目都有独立的 README.md 说明
4. 建议按照分类顺序学习
